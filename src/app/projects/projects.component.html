Academic Projects

Crowd Counting (Project in Mastek Pvt Ltd)

November 2020 - April 2021

Team Size: 4 | Role: Software Developer

Developed a web application for accurately counting people in a mall using real-time camera footage. Leveraged YOLO for human detection and utilized Flask and Angular for the web interface. Implemented custom algorithms to manage overlapping camera regions and differentiate between mannequins, images, and actual humans. The project provided valuable experience with machine learning frameworks and has potential applications in safety and retail analytics.

Tech Stack: YOLO, TensorFlow, Python, Flask, Angular

Gym Manager Application (S.I.E.S GST, Nerul)

August 2020 - December 2020

Team Size: 3 | Role: Team Lead & Software Developer

Led the development of a comprehensive web application to manage gym memberships and client interactions.
